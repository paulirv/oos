# Guideline 3.3: Interoperability

**Principle:** [Machine-Readable](../principles/machine-readable.md)

Multiple implementation formats are supported.

## Intent

The web is not one technology. HTML pages, API responses, static site generators, and CMS platforms all need viable implementation paths. The spec defines a single data model expressed in multiple formats.

## Success Criteria

| ID | Level | Criterion |
|----|-------|-----------|
| 3.3.1 | L2 | When multiple formats are used, declarations MUST convey the same origin information |
