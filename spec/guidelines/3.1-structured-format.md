# Guideline 3.1: Structured Format

**Principle:** [Machine-Readable](../principles/machine-readable.md)

Declarations use defined schemas.

## Intent

Origin declarations use a standardized schema that machines can parse without ambiguity. Free-text declarations ("this was written with AI help") don't count — the data must be structured.

## Success Criteria

| ID | Level | Criterion |
|----|-------|-----------|
| 3.1.1 | L1 | Declaration MUST use a recognized OpenOrigin format |
| 3.1.2 | L2 | Declaration MUST validate against the OpenOrigin schema |
