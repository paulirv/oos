# Guideline 4.2: Minimal Burden

**Principle:** [Practical](../principles/practical.md)

L1 conformance requires minimal effort.

## Intent

The barrier to entry must be low enough that any publisher — from a solo blogger to an enterprise CMS — can achieve L1 conformance with trivial effort. One meta tag. One header. One frontmatter field.

## Success Criteria

This guideline has no per-content success criteria. The minimal burden property is a design property of the specification itself. See [Conformance: Design Properties](../conformance.md#design-properties).
